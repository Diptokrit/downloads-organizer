# Downloads Organizer Automation

A Python automation script that organizes files automatically into categorized folders.

## Features
- Organizes files by type
- Handles duplicate filenames automatically
- Creates folders automatically
- Sends unknown files to Other
- Ignores folders

## Categories
- Images
- Videos
- Documents
- Code
- Archives
- Audio
- Other

## Tech Stack
- Python
- pathlib
- shutil

## Run
python organizer.py
